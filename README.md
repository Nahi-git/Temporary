Please have google chrome when using this. As webgazer works through chrome.

When using the app through unity editor, Please run the calibration scene first. Once the calibration scene is running, a chrome window will pop up and ask you to calibrate your gaze through your webcam.

After calibrating in the chrome window, please have this tab running in the background. DO NOT CLOSE OR MINIMISE as it will stop giving gaze feedback.

Now calibrate again within the unity calibration scene.

After both calibration, use the following hotkeys to move to your desired scene.

GazeTouch Demo: Q
GazeOnly Demo: W
TouchOnly Demo: E

GazeTouch: A
GazeOnly: S
TouchOnly: D

You can use these hotkeys within any scene.

After completing the user study scenes, The CSV file containing the data of that scene will spawn in the GazeTouchLogs folder.
